# Getting Started

## Local Setup

Required tools:

- Java 21 or newer
- Maven 3.9+
- Docker Desktop
- GitHub CLI

## First Workflow

1. Read the active sprint plan.
2. Pick a Jira story from `docs/jira`.
3. Create a feature branch from `develop`.
4. Implement the change with tests.
5. Open a pull request.
6. Complete code review and CI checks.
7. Merge back into `develop`.
